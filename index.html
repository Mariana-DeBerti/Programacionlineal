¿Qué es la programación lineal?

Es una técnica matemática que permite optimizar (maximizar o minimizar) una función objetivo sujeta a restricciones.

Se usa para tomar decisiones eficientes cuando los recursos son limitados

Se basa en lo siguiente:

  1. Definir el problema, lo que queremos optimizar (Mazimizar ganancias, minimizar costos, optimizar el uso de recursos)
  2. Identificar las variables de decisión, son las incognitas del problema (Representa lo que podemos decidir)
  3. Formular la función objetivo, es la expresión matemática que queremos optimizar(debe ser lineal)
  4. Estableces las restricciones, representan los límites reales del problema (los recursos limitados)
  5. Condiciones de no negatividad (las variables no pueden ser negativas)
  6. Resolver el modelo (Método gráfico, método simplex, software)
  7. Analizar e interpretar la solución (¿tiene sentido?¿Cumple restricciones?¿Es aplicable en la prática?

¿Para qué se utiliza?

  • Tomar decisiones óptimas cuando hay recursos limitados y varias alternativas
  • Elegir la mejor opción usando matemáticas
  • En industrias se utiliza para la planificación de producción y asignación de materias primas
  • En logística se utiliza para la optimización de rutas y distribución de mercadería

Ejemplo

Planificación de siembra

Un productor dispone de 100 hectáreas y un presupuesto de $60.000 para sembrar maíz y soja

Datos:

Cultivo Rinde(tn/ha) Precio ($/tn) Costo ($/ha)
Maíz 8 150 800
Soja 3 350 500

Ingreso y margen por hectárea

Ingreso por ha

  • Maíz: 8*150 = $ 1200
  • Soja: 3*350 = $1050

Margen bruto por ha

Ingreso - costo:

  • Maíz: 1200 - 800 = 400 $/ha
  • Soja: 1050 -500 = 550 $/ha

Formulación matemática

Variables

  • x = ha de maíz
  • y = ha de soja

Función objetivo: Maximizar margen

Z= 400x + 550 y

Restricciones

Tierra disponible

x + y <= 100

Presupuesto

800x + 500y <= 60000

No negatividad

x,y >= 0

Usaremos valores aleatorios dentro de estos rangos

Resolución

Método gráfico

Simplificamos presupuesto: 8x + 5y <= 600

Evaluación

Punto Maíz Soja Margen
A 0 100 55.000
B 75 0 30.000
C 33,3 66,7 51.685

Solución óptima: sembrar 100 ha de soja

Interpretación

  • La soja tiene mayor margen por hectárea
  • El presupuesto limita la expansión del maíz
  • El modelo sugiere especialización según rentabilidad, sin considerar otras variables como rotación de cultivo, salud del suelo, etc.
