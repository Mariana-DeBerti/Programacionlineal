¿Qué es la programación lineal?

Es una técnica matemática que permite optimizar una función objetivo sujeta a restricciones.

Se usa para tomar decisiones eficientes cuando los recursos son limitados.

¿En qué consiste?

  1. Identificar variables de decisión
  2. Establecer restricciones
  3. Resolver el modelo
  4. Maximizar beneficios o minimizar costos
  5. Planificación productiva
  6. Optimización logística

Ejemplo: Planificación de siembra

Un productor dispone de 100 hectáreas y un presupuesto de $60.000. Quiere saber si le combiene sembrar maíz o soja.

Cultivo Rinde Precio Costo
Maíz8150800
Soja3350500

Margen por hectárea

  • Maíz: 8×150 − 800 = $400
  • Soja: 3×350 − 500 = $550

Formulación matemática

Variables: x = maíz, y = soja

Maximizar: Z = 400x + 550y

Restricciones:

x + y ≤ 100

800x + 500y ≤ 60000

Cálculo del punto óptimo

x + y = 100

800x + 500y = 60000

y = 100 − x

800x + 500(100 − x) = 60000

300x = 10000 → x = 33,3

y = 66,7

Intersección: (33,3 ; 66,7)

Evaluación de soluciones

Solución óptima: 100 ha de soja.

Región factible

Código utilizado


Región factible:

Óptimo:
[{x:0,y:100}]

Interpretación

  • La soja tiene mayor margen de ganancia.
  • El presupuesto limita la expansión del maíz.
  • El modelo no considera rotación de cultivos ni salud del suelo.
